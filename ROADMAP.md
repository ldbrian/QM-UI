# QM-UI 产品路线图与技术规划 (Roadmap)

## 📍 当前版本：V1.0 (预售 MVP 版)
**核心定位：** 极速录入心流与防爆雷底座。
**已实现能力：**
- [x] **纯键盘二维引擎 (2D Keyboard Nav)：** 支持 Enter 向下、Tab 向右、方向键无死角穿梭，支持只读/编辑单元格的焦点切换。
- [x] **扫码连录心流：** 自动新增行 -> 自动 Focus -> 自动 Select 全选，实现零鼠标干预的盲打录入。
- [x] **错题状态机 (Error Docking)：** 失焦静默校验，错误就地标红，提供“点击聚拢吸底”与“自动平滑滚动”的高级防呆交互。
- [x] **引擎完全解耦：** `BaseGridEngine` 与业务层完全分离，支持 `features` 开关灵活配置（十字星、键盘导航、Dock区）。

---

## 🚀 未来里程碑：V1.5 (性能与数据清洗增强)
*（预售完成后首批攻克的技术债）*
- [ ] **虚拟滚动引擎 (Virtual Scrolling)：** 剥离原生 DOM 渲染，引入按需渲染机制，支撑 10 万行级别数据无损耗键盘穿梭。
- [ ] **智能剪贴板解析 (Excel Paste)：** 劫持全局 `Ctrl+V`，支持将外部 Excel 块状数据直接粘贴映射并静默校验。
- [ ] **复杂列宽状态机 (Column Resizing & Pinning)：** 支持表头拖拽调整列宽、首列冻结，并结合 LocalStorage 实现用户习惯记忆。

---

## 🌟 创世生态：V2.0+ (复合业务组件集)
*（向创世会员兑现的 50% 折扣高级组件库）*
- [ ] **无限极树状 BOM 表 (Tree-BOM Grid)：** 专为物料清单与组织架构设计，支持跨层级拖拽与节点懒加载。
- [ ] **高密交叉筛选表格 (Dense Filter Grid)：** 应对海量数据清洗与多条件查询，内置记忆化布局方案。
- [ ] **多步分段表单与侧边抽屉 (Form UX)：** 防误触、高密度的 B 端表单输入体验。
